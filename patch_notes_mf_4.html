MF: Voice of the Martian Updates – February 4th Patch Notes

February 4, 2021July 26, 2022

New this week in Mars Frontier:

Voice of the Martian Updates

Heads up, Martians! This week we’re bringing you a couple quality-of-life updates based on your player feedback from the “Voice of the Martian” Trackers. Don’t see the update you were hoping for in this week’s releases? Fear not, as there is still plenty more to come!

For this week, the Upgrade Information bar that lists the benefits of an upgrade has had its scroll buttons fixed. Mouse-wheel scrolling is now supported.

Chat copy-to-clipboard

Chat messages have been updated. Was that last chat so funny you want to save it forever? Confused if someone using another language is talking trash or feeling friendly? All messages now have a “Copy to Clipboard” feature. You can paste into your favorite translation program or your secret repository of clan messages.

May the ability to translate chat messages bring more understanding and enjoyment of the game rather than less!

New “Repair Base” button

You may notice a new wrench button on your controls. This new repair base feature will allow you to bring back the repair for gold window if you chose “repair slowly”.

It will also inform you via tooltip/hovertext how much alloy it will take to fully repair your base.

Best of all, once your buildings hit the free repair threshold, the wrench will turn green, indicating that you can repair all buildings for free with a single click. No more visiting each building in your base and clicking “speed up” when they hit five minutes.

MF: “Friends Bar” Changes – January 21st Patch Notes

January 21, 2021July 26, 2022

New this week in Mars Frontier:

“Friends Bar” Changes

Heads up, Martians! We have a small update this week.

Due to the changes in the Facebook API, all friend recruitment quests haven’t been working as intended. We’ve decided to disable these quests. If you completed one of these quests, you should see an achievement in the Legacy category:

“Congrats on being such a friendly person!”

For all Facebook games: you will no longer see the friend bar if you have no Facebook friends in game. If you still have the friend bar, you will no longer be prompted or be able to add new friends.

  • Firestrike is not affected as it primarily uses the Battlehouse login system.
  • Thunder Run players using the Kongregate platform are not affected.
  • Battlefront Mars players using the Battlehouse login system are not affected.

In the future, we do plan on bringing back the “friends” bar, but more as a “clan bar.” We’re open to player feedback on how they’d like to see this implemented.

If you have any suggestions about this, or any general feedback you’d like to share, please let us know over the “mars-frontier” channel within our Thunder Run Discord Server.
